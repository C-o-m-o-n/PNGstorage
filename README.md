# PNGstorage

encrypt files into PNG images that appear to be white noise.

## Usage

```bash
poetry install

poetry run python PNGstorage.py
```

## Example

```bash
poetry run python PNGstorage.py

1. Encode
2. Decode
3. Exit

Choice: 1

Select file (Select File (current path: /Users/Kokonico/PycharmProjects/PNGstorage): README.md

Done!
```
